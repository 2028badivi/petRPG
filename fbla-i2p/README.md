# PetRPG: Future Planner (FBLA Intro to Programming)

PetRPG is a professional-grade virtual pet simulation designed to teach financial responsibility and logical planning through code. Built specifically for the FBLA 2025-2026 Intro to Programming topic.

## 🚀 Features

- **Logic-Based Planning**: Use Google Blockly to "code" your pet's daily routine.
- **Economic Simulation**: Balance a budget, earn a salary, and track the "Cost of Care" in real-time.
- **Pixel Art RPG World**: A beautiful top-down game engine built with Phaser 3, inspired by Champion Island.
- **Dynamic Vitality**: Manage Hunger, Happiness, Health, Hygiene, and Energy.
- **Leveling System**: Grow from a novice to a master pet owner with increasing complexity.
- **Glassmorphism UI**: A modern, premium dashboard for a sophisticated user experience.

## 🛠️ Technology Stack

- **Frontend**: Vanilla JavaScript, Vite
- **Game Engine**: [Phaser 3](https://phaser.io/)
- **Logic Engine**: [Google Blockly](https://developers.google.com/blockly)
- **Styling**: Modern CSS3 (Glassmorphism, Grid, Flexbox)
- **Storage**: LocalStorage API

## 📋 How to Use

1. **Name Your Pet**: Edit the pet's name to personalize your experience.
2. **Plan Your Routine**: Drag blocks from the "Logic Planner" Categories:
   - **Pet Care**: Feed, Play, Clean.
   - **Health**: Visit the Vet.
   - **Economics**: Go to Work to earn currency.
3. **Check Costs**: The "Projected Cost" updates as you add or remove blocks.
4. **Run Simulation**: Click "Run Simulation" to execute your plan. Watch your stats and currency update!
5. **Level Up**: Keep your pet happy and your budget balanced to reach higher levels.
